An Oregon Climber's Guide Application

This application provides users with top rope and bouldering climbing locations across the state of Oregon. Users can filter the routes displayed, read about existing sites and add comments, and add new routes. It was built with PostgreSQL, Java, JQuery, Bootstrap, and the Google Maps API when enrolled in UW Madison's Spatial Web and Mobile Programming course (GEOG 576).
